Inkskepu
====

## 説明
inkscapeに使いにくさを感じた衝動で作り始めた。  


変数とか関数とか使えるようにしたい(願望)。  
GIMPやBlenderみたいにGUIなしでpythonスクリプトとか使えるようにしたい(願望)。  


svgを独自拡張してアニメーションさせて動画も作れるようにしたい(願望)。  
木構造を描画するクラス、魔方陣を描画するモジュール、入れたり入れなかったりしたい(願望)。  
画面に表示するペインを好きに移動したり、ペインモジュールを入れたり入れなかったりしたい(願望)。  

## 必要な環境
java(古すぎないもの)

## 使い方
compile.batでコンパイル,start.batで起動。最終的には `実行ファイル ファイル オプション` で起動できるようにしたい。
